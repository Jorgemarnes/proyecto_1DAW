# proyecto_1DAW - Music Catalog SSG Project 🎸

## Project Overview
This project is a Static Site Generator (SSG) that builds a music store catalog using Python, PostgreSQL, and VirtualBox.

## Architecture
The system consists of:
1. **Host Machine**: Browser access.
2. **VM Web (Guest 1)**: Runs Nginx/Apache serving static HTML.
3. **VM Database (Guest 2)**: Runs PostgreSQL (Port 5432).

## Project Summary

**Music Store** is a static website generator project based on music data obtained from the **iTunes Search API**.

The project simulates a fictional platform where users can browse albums and songs and buy music licenses. The data is processed with Python, stored in PostgreSQL and then used to generate static HTML files served by Nginx.

This documentation explains the requirements, database structure, design, infrastructure, scraping process, setup steps and security plan of the project.
